# 🍽️ Khana Khanjana - Food Blog Website

Khana Khanjana is a dynamic **food blogging platform** where users can explore, share, and post their favorite recipes. It allows users to register, log in, and contribute **veg & non-veg food blogs** with images, descriptions, and step-by-step cooking instructions.

## ✨ Features:
- **User Authentication:** Secure **Signup & Login** system.
- **Dynamic Blog Posting:** Users can add their **own food blogs** under Veg or Non-Veg categories.
- **Interactive UI:** Blogs appear as **cards** with a seamless interface.
- **Image Uploading:** Upload and showcase your dish with **high-quality images**.
- **Database Integration:** Stores user & blog details securely in an **SQL database**.

## 🛠️ Tech Stack:
- **Frontend:** HTML, CSS, JavaScript (AJAX for dynamic UI updates)
- **Backend:** PHP, MySQL (for database handling)
- **Hosting:** XAMPP (for local testing)
